## Asset Descriptions

### Screenshots

1. **screenshot_1.png** - Main AR Experience

   - Show the primary use case: AR avatar appearing on a product poster
   - Highlight the seamless integration of digital avatar with physical world
   - Include UI elements like status indicators

2. **screenshot_2.png** - Camera Interface

   - Display the camera viewfinder with targeting reticle
   - Show product recognition in progress
   - Include navigation elements and settings

3. **screenshot_3.png** - Avatar Speaking
   - Capture the avatar in mid-speech with lip-sync animation
   - Show subtitle/closed caption overlay
   - Demonstrate interactive elements

### App Icon

- **app_icon.png**
- Clean, recognizable logo representing AR/AI technology
- Simple design that works at small sizes
- Brand colors: Blue (#2196F3) and Green (#4CAF50)

### Feature Graphic

- **feature_graphic.png**
- Promotional banner for Play Store listing
- Showcase key app features visually
- Include tagline: "Transform Static Posters Into Interactive Experiences"
